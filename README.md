# 📊 Habit Tracker (Excel)

An Excel-based habit tracking system designed to monitor daily habits and analyze consistency over time.

This project transforms simple daily inputs (checkboxes) into structured insights using Excel formulas, conditional formatting, and progress indicators.

The tracker helps users visualize their habit completion, measure progress toward monthly goals, and better understand behavioral patterns.

# 🎯 Project Goals

This project aims to:

* Track daily habit completion for an entire month
* Measure consistency using completion metrics
* Provide visual progress indicators
* Practice data analysis techniques using Microsoft Excel

---

# ✨ Features

Current features include:

* ✔️ Daily habit tracking using checkboxes
* 📅 Monthly tracking layout
* 📊 Progress bars for habit completion
* 🔢 Automatic counting of completed habits
* 📈 Progress comparison between **Goal vs Actual**

---

# 🧠 How It Works

Each habit is tracked daily using checkboxes.

Checked boxes represent completed habits and are automatically converted into data used to calculate:

* Total completed days per habit
* Monthly completion percentage
* Progress toward the monthly goal

These calculations power the progress bars and analytics displayed in the spreadsheet.

---

# 📥 How to Use

1. Download the Excel file from this repository
2. Open it in Microsoft Excel
3. Check the boxes for habits you complete each day
4. The progress bars and totals will update automatically
